SET METHODS 

- add()

- update()

- remove()

- discard()

- clear() : it empties the set

- del() : it will delete the set completely

- intersection() 

- intersection_update() : this method removes the items that is not present in both sets

- union()

- copy()

- isdisjoint() : returns True if none of the items are present in both sets,otherwise it returns False

- issubset() : method returns True if all items in the set exists in the specified set, otherwise it returns False.

- symmetric_difference() : method returns a set that contains all items from both set, but not the items that are present in both sets.

- symmetric_difference_update() : method updates the original set by removing items that are present in both sets, and inserting the other items.

Difference between remove discard is
      remove() : if the item to remove doesn't exist,remove() will raise an error
      discard() : if the item to remove doesn't exist,discard() will Not raise an error

you can also use the pop() method to remove an item , but this method will remove a random item , cannot be sure about which item that gets removed .
